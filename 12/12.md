## очередное задание, тест на дауна

     НАЧАЛО
     ПОКА нашлось (78) ИЛИ нашлось (888)
       ЕСЛИ нашлось (78)
         ТО заменить (78, 8)
       КОНЕЦ ЕСЛИ
       ЕСЛИ нашлось (888)
         ТО заменить (888, 7)
       КОНЕЦ ЕСЛИ
     КОНЕЦ ПОКА
     КОНЕЦ

```python
def f(s):
    while '78' in s or '888' in s:
        if '78' in s:
            s = s.replace('78', '8', 1)
        if '888' in s:
            s = s.replace('888', '7', 1)
    return s

cnt = 0
for n in range(1000, 2001):
    s = '7' + '8' * n
    ans = f(s)
    summ = sum(map(int, ans))
    if summ == 16:
        cnt += 1
print(cnt)
```
